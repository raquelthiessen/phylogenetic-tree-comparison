See https://github.com/adamallo/SimPhy/wiki/Manual and online Supplementary Material of ASTRAL-Pro paper

"A typical simulation scenario would start simulating a species tree, followed by the simulation of a 
number of locus trees and the subsequent simulation of a gene tree per locus tree"


Simphy command for default parameters:

ASTRAL-Pro
simphy -sl f:25 -rs 50 -rl f:1000 -rg 1 -sb f:0.000000005 -sd f:0
-st ln:21.25,0.2 -so f:1 -si f:1 -sp f:470000000 -su ln:-21.9,0.1
-hh f:1 -hs ln:1.5,1 -hl ln:1.551533,0.6931472 -hg ln:1.5,1 -cs 9644
-v 3 -o default -ot 0 -op 1 -lb f:0.00000000049 -ld f:0.00000000049
-lt f:0


-sl f:25 			(*) number of taxa
-rs 50 				(int) number of species tree replicates (i.e., study replicates)
-rl f:1000			(*) number of locus trees per species tree
-rg 1 				(int) number of gene trees per locus tree
-sb f:0.000000005 		(*) speciate rate (events/time unit)
-sd f:0				(*) extinction rate (events/time unit)
-st ln:21.25,0.2 		(*) species tree height (time units)
-so f:1 			(*) ratio between ingroup height and the branch from the root to the ingroup
				if this parameter is not set the outgroup is not simulated
-si f:1 			(*) number of individuals per species
-sp f:470000000 		(*) tree-wide effective population size
-su ln:-21.9,0.1		(*) tree-wide substitution rate
-hh f:1 			(*) gene-by-lineage-specific locus tree parameter (to use with hg parameter)
-hs ln:1.5,1 			(*) species-specific branch rate heterogeneity modifiers
-hl ln:1.551533,0.6931472	(*) gene-family-specific rate heterogeneity modifiers
-hg ln:1.5,1 			(*) gene-by-lineage-specific rate heterogeneity modifiers
-cs 9644			(int) random number generator seed
-v 3 				(0-6) verbosity level
-o default 			output parameters
-ot 0 				(bool) determines whether the species and locus tree branches are written in 
				number of generations (0) or time units(1)
-op 1 				(bool) activates logging of sampled options
-lb f:0.00000000049 		(*) locus tree parameters: duplication rate (events/generationg)
-ld f:0.00000000049		(*) locus tree parameters: loss rate (events/generation)
-lt f:0				(*) locus tree parameters: horizontal gene transfer rate (HGT) (events/generation)


* = sampling notation
Distribution		Code 	Parameter 1		Parameter 2
Point (fixed value)	f	value
Lognormal		ln	location		scale
